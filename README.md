# Rock-Paper-Scissors

A beautiful, interactive web-based Rock-Paper-Scissors game built as part of The Odin Project curriculum.

## Features

- **Customizable Rounds:** Choose exactly how many rounds you want to play before the game starts.
- **Interactive UI:** Smooth hover effects, custom typography, and a modern modal overlay for the gameplay.
- **Score Tracking:** Live updates of both the Player and Computer scores.
- **Announcements:** Immediate feedback on round winners and a final match declaration when all rounds are completed.

## Technologies Used

- HTML5
- CSS3 (Flexbox, transitions, custom styling)
- Vanilla JavaScript (DOM manipulation, event delegation)

## How to Play

1. Open `index.html` in your web browser.
2. Enter the number of rounds you want to play and click **Submit**.
3. A modal will pop up with the game board. Click on your choice—Rock (🪨), Paper (🗞️), or Scissors (✂️).
4. The scoreboard will update after each pick. After the final round, the overall winner will be displayed!
